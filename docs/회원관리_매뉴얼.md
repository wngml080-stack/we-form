## 1. 신규 회원 상담 기록지

> 첫 상담 시 회원의 배경 정보와 목표를 파악하기 위한 기록지

### 1-1. 기본 정보
| 항목 | 설명 | 입력 예시 |
|------|------|-----------|
| 회원명 | 회원 이름 | 홍길동 |
| 연락처 | 휴대폰 번호 | 010-1234-5678 |
| 담당 트레이너 | 배정된 트레이너 | 김트레이너 |
| 상담 유형 | 신규/재등록/이관 등 | 신규 |
| 최초 미팅일 | 첫 상담 날짜 | 2025-01-02 |

### 1-2. 방문 경로
회원이 센터를 알게 된 경로를 체크합니다.

- **네이버 플레이스** - 네이버 지도/플레이스 검색
- **인스타그램** - SNS 광고 또는 계정
- **블로그** - 블로그 후기/정보
- **지인 소개** - 소개자 이름 기록 필수
- **워크인** - 직접 방문
- **기타** - 기타 경로 (직접 입력)

💡 **추가 질문**: "저희 센터의 어떤 점이 마음에 드셨나요?" (매력 포인트 파악)

### 1-3. 운동 경험
이전 운동 경험을 파악합니다.

| 운동 유형 | 경험 여부 | 기간(개월) |
|-----------|-----------|------------|
| 헬스장 | ✅/❌ | - |
| 필라테스/요가 | ✅/❌ | - |
| 1:1 PT | ✅/❌ | - |
| 홈트레이닝 | ✅/❌ | - |
| 기타 | ✅/❌ | - |
| 운동 경험 없음 | ✅/❌ | - |

### 1-4. 중도 포기 이유 (해당 시)
과거 운동 중단 사유를 파악합니다.

- ❌ 변화가 없어서
- ❌ 관절/근육 통증
- ❌ 다이어트 실패
- ❌ 자신감 부족
- ❌ 시간 부족
- ❌ 비용 문제
- ❌ 트레이너와 안 맞음
- ❌ 기타

### 1-5. 신체 기능 & 생활 습관

#### 통증 부위 체크
| 부위 | 해당 | 강도(1-5) | 진단명 |
|------|------|-----------|--------|
| 목/어깨 (승모근, 두통) | ✅/❌ | 1-5 | 예: 일자목 |
| 허리 (디스크, 골반 틀어짐) | ✅/❌ | 1-5 | - |
| 무릎/발목 (소리, 통증) | ✅/❌ | 1-5 | - |
| 손목/팔꿈치 | ✅/❌ | 1-5 | - |
| 기타 | ✅/❌ | 1-5 | - |

#### 병력 및 치료
- **과거 병력**: 유/무, 진단명 기록
- **현재 치료 중**:
  - 물리치료
  - 도수치료
  - 약물 복용
  - 기타

#### 생활 패턴
| 항목 | 입력값 |
|------|--------|
| 하루 앉아있는 시간 | __시간 |

| 회당 음주량 | __잔 |

- ❌ 규칙적

- ❌ 폭식

### 1-6. 목표 설정

#### 다이어트 목표
| 항목 | 값 |
|------|-----|
| 현재 체중 | __kg |
| 목표 체중 | __kg |
| 감량 목표 | __kg |
| 목표 달성일 | YYYY-MM-DD |
| 과거 다이어트 시도 횟수 | __회 |
| 인생 최저 체중 | __kg (__년 전) |
| 선호 방법 | 운동 / 식단 / 둘 다 |

#### 재활/체형 목표
- 거북목 / 라운드숄더 / 디스크 / 골반 불균형 / 척추측만 / 무릎통증 / 기타
- 의사 진단 여부
- 일상생활 영향도 (1-5점)
- 기대 사항

#### 근력 목표
- 세부 목표: 전반적 체력 / 바디프로필 / 3대운동 / 벌크업 / 스포츠 퍼포먼스 / 기타
- 현재 3대 중량 / 목표 중량
- 운동 경력

#### 습관 형성 목표
- 세부 목표: 규칙적 운동 / 수면 개선 / 건강한 식단 / 스트레스 관리 / 자세 교정 / 기타
- 현재 습관 / 목표 습관

#### 운동 동기
- 특별한 이벤트 (날짜 기록)
- 건강 경고
- 자존감
- 심한 통증
- 체력 저하
- 기타 / 구체적 이유

## 기록지 활용 가이드

### 회원 여정별 기록지 사용 흐름

```
신규 상담 → 상담 기록지 작성
    ↓
OT 진행 → OT 기록지 작성
    ↓
PT 시작 → PT 기록지로 지속 관리
    ↓
재등록/재측정 시 → 인바디 업데이트, 비교 분석
```

### 권장 기록 주기

| 항목 | 주기 | 비고 |
|------|------|------|
| 세션 기록 | 매 PT 후 | 당일 기록 권장 |
| 인바디 측정 | 4주마다 | 동일 조건(시간, 복장) |
| 둘레 측정 | 4주마다 | 인바디와 함께 |
| 비포/애프터 사진 | 4주마다 | 동일 장소, 조명, 포즈 |
| 체력 테스트 | 8주마다 | 진전도 확인용 |

### 기록 작성 팁

1. **상담 기록지**: 회원의 말을 최대한 그대로 기록하여 니즈 파악
2. **OT 기록지**: 숫자는 정확하게, 평가는 근거와 함께 기록
3. **PT 기록지**: 매 세션 즉시 기록하여 누락 방지

## 자주 묻는 질문 (FAQ)

**Q. 기록지를 작성하다가 저장하지 않고 나가면?**
A. 자동 저장되지 않으므로 반드시 저장 버튼을 클릭하세요.

**Q. 회원 정보를 수정하려면?**
A. 해당 기록지를 열어 수정 후 저장하면 됩니다.

**Q. OT 없이 바로 PT를 시작하는 회원은?**
A. PT 기록지에서 "처음 만난 날 측정" 섹션에 기본 측정값을 기록합니다.

**Q. 트레이너가 변경되면?**
A. 기본 정보의 담당 트레이너를 변경하고, 인수인계 내용을 댓글에 기록합니다.
